## Super Bees - AI Bee-Soultion 프로젝트

# **프로젝트 개요**
 - 사용자들이 일상과 고민들을 공유하고 소소한 힐링과 동기부여를 얻어 갈 수 있는 웹 서비스

  - 익명의 사용자가 고민을 올리면 꿀벌들이 협동하듯 사용자들이 솔루션을 주고 인공지능 Superbee가 최적의 솔루션을 제시 해주는 웹 서비스 개발

## 목표
